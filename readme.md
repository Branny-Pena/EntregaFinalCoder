Instalar django

```shell
python -m pip install django
```

Aplicar cambios en base de datos

```shell
python manage.py makemigrations
python manage.py migrate
```

Iniciar servidor

```shell
python manage.py runserver
```

Crear usuario administrador

```shell
python manage.py createsuperuser
```
